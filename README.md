# 剣道選手の成長促進Webアプリケーション  「Kendo Life」

## 概要
Kendo Lifeは、剣道選手が日々の稽古や試合を振り返り、成長につなげることを目的とした
Webアプリケーションである。  
試合スコアや得点の流れ、スケジュール、日々の課題を一元管理することで、
剣道生活をより記録しやすく、振り返りやすくすることを目指して開発した。

## 誰のために？
日々の稽古や試合を通して成長を目指す剣道選手を対象としている。  
特に、試合内容や得点の流れを記録・振り返りたい学生剣士を想定している。

## 何ができる？
本アプリケーションには、以下の5つの機能を実装している。

- **スケジュール管理機能**：試合や稽古の予定を登録・管理できる  
- **ToDoリスト管理機能**：日々の課題や目標を整理できる  
- **試合スコア記録機能**：試合結果や得点履歴を記録・可視化できる  
- **掲示板機能**：チーム内での情報共有を行える  
- **プロフィール管理機能**：選手情報を登録・編集できる  

## 技術・開発概要
Ruby on Railsを用いて開発したWebアプリケーションである。  
バックエンドはRails、フロントエンドはHTML・CSS・JavaScriptで構成し、
Renderを利用してデプロイしている。

## サービスを作ったきっかけ
小学校から高校までの剣道経験を通じて、剣道はマイナースポーツであり、
記録や振り返りを支援するデジタルサービスが少ないと感じていた。  
実際、試合のスコア記録は紙で行われることが多く、
紛失のリスクや過去の試合を振り返りにくい点が課題であった。

そこで、剣道経験のある友人8名にヒアリングを行った結果、
同様の不便さを感じていることが分かり、
これらの課題を解消するサービスとして本アプリケーションを開発した。

## 大変だったこと
- 剣道特有の複雑なルールや、厳格に定められたスコア表の記入方法を、
  アプリケーションの仕様として整理し、コードに落とし込む点に苦労した。
- 試合の流れや得点順を正しく再現するため、
  データ構造や表示ロジックを試行錯誤しながら実装した。

## 工夫したこと
- いつでもどこでも記録できることが使いやすさにつながると考え、
  スマートフォンでの利用を想定したレスポンシブ対応を行った。
- 特にスコア記録機能では、一般的に横長で表現される剣道のスコア表を、
  スマートフォン画面に収まる縦型レイアウトへ再設計した。
- 本来の形式とは異なる表示になるため、初めて利用するユーザーでも
  直感的に理解でき、見やすく使いやすいレイアウトになるよう意識した。
- CSS Gridを用いて、ポイント履歴を縦（選手）×横（時系列）で表現した。
- HTMLとCSSを分離し、保守性を意識した構成とした。
- `content_for` を用いて、ページごとに必要なCSSのみを読み込む設計にした。
- RuboCopを導入し、Lintエラーを修正しながら開発を行った。
